# Eumnesia Labs — brand assets

All current logo files. SVG = master (scales to any size); PNG = ready-to-upload.
Palette: ink `#0D1219` · amber `#DFAC4C` · slate `#232D3B` · text `#E9EDF2`.
Full brand rules: [`../BRAND_BRIEF.md`](../BRAND_BRIEF.md) (Part A).

| file | mark | story | use for |
|---|---|---|---|
| `mark-e.svg` / `mark-e-512.png` | **E-mark** (three strokes, middle amber) | of everything, the one stroke remembered | **primary** — GitHub org avatar, favicon, profile pictures |
| `logo-blocks.svg` / `-512.png` / `-64.png` | **Blocks** (3×3 grid, one amber) | nine blocks, one remembered — block-sparse selection | patterns, textures, secondary mark |
| `mark-infinity.svg` / `-512.png` | **E-infinity** (loop dissolving into blocks) | memory loops forever; the far side dissolves | hero graphics, deck covers, brand moments |

Rules: never stretch, never recolor outside the palette, never put text inside the mark
(wordmark "EUMNESIA LABS" — space, no hyphen — is typeset separately). AI-generated
draft explorations can be dropped in a `drafts/` subfolder; they are not official marks.
